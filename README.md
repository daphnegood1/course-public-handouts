## 公開原則

這個 repo 只放已審核、可公開給學生閱讀或下載的內容。

不放：

- 老師備課版
- source map
- 原始逐字稿
- 私有 Google Drive / GitHub / Obsidian 連結
- API key、token、帳號資訊
- 未審核草稿
